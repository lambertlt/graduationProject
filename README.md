# graduationProject

一个前后端项目——关于音视频的app

前端框架：vue-element-admin

后端框架：springBoot + spring security + spring data jpa

目录结构：

```shell
.
├── README.md 
├── appBackend  后端项目文件夹
└── appWeb      前端项目文件夹
```

## project author

由Lambert、liulinboyi共同完成毕业设计。

## appBackend 后端项目部署步骤

1. 将appBackend下的`app.sql`在mysql命令行运行
2. 用 eclipse 或 idea 运行 appBackend 目录下的项目运行即可
